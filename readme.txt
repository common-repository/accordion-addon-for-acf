Accordion Addon plugin allows user to use the accordion type field using the ACF plugin.

== Description ==

The Accordion Type field is used to group fields into accordion sections.

When editing a field group, be aware that all fields following the Accordion Type field (or until another Accordion Type field is defined) will be grouped using the Accordion Type field label as the Accordion Type heading.

**Important Note:** Accordion Addon for ACF is built on top of Advanced Custom Fields. This plugin requires an ACF plugin to function. Make sure you have ACF installed and activated.

== Installation ==

This plugin can be installed directly from your site.
1. Login and navigate to Plugins â†’ Add New.
2. Type "Accordion Addon for ACF" into the Search and hit Enter.
3. Locate the Accordion Addon for the ACF plugin in the list of search results and click Install now.
4. Once installed, click the Activate link.

It can also be installed manually.

1. Download the Accordion Addon for the ACF plugin from WordPress.org.
2. Unzip the package and move it to your plugins directory.
3. Login to the WordPress and navigate to the Plugins screen.
4. Locate the Accordion Addon for ACF in the list and click the Activate link.

== Frequently Asked Questions ==

= Do I need a specific theme to use this plugin?

No, this will be work for all themes with Accordion ACF.

= Is Advanced Custom Fields required to use this plugin?

Yes, Advanced Custom Fields or Advanced Custom Fields Pro is required to use this plugin.
